# Sales-Dashboard
A Sales Dashboard is a visual report that shows key sales data like total sales, profit, profit margin and product sold. It helps businesses track sales trends, compare performance, and make better decisions quickly using charts, KPIs, and filters.

## Dataset used
<a href="https://github.com/SathyaPBI2026/Sales-Dashboard/blob/main/Sales%20Analysis%20Report%2011.xlsx">Dataset</a>

## Questions (KPIs)
•	What is the total sales amount? 

•	What is the total profit? 

•	What is the profit margin percentage? 

•	What is the total product sold amount? 

•	Comparing sales vs sales last year by product name?

•	Comparing sales with previous year sales by year?

•	Comparing sales by city?

•	Comparing sales vs sales last year by customer name?

•	Comparing profit, profit last year and profit margin by channels?

•	Are sales increasing or decreasing over time?

•	Which products give the highest profit margin?

•	Which city has the highest profit?

## Process
•	Import the data from the source files.

•	Cleaning it, to remove duplicate, errors, null values and format columns using power query.

•	Data model-Built relationship between tables

•	Add a calendar/date table for time intelligence analysis.

•	Write DAX Measures: Create calculations like Total sales, Profit, Profit margin.

•	Built visuals like charts, cards, tables, KPIs.

•	Add filters, interactions, design dashboard, analyze insights and publish it.
